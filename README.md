### Usage example

Install dependents
```
pip3 install -r requirements.txt
```

Run script

```

python3 vpcaws.py \
--client_network=<specific_ippool> \
--main_vpc_id=vpc-<main_VpcId> \
--region_name=<region_name> \
--aws_access_key_id=<aws_access_key_id> \
--aws_secret_access_key=<aws_secret_access_key>

```
